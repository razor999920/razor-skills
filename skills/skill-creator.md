# skill-creator

**Group:** Skill & plugin authoring

## What it does
Create new skills from scratch, edit and improve existing ones, and measure skill performance: run evals, benchmark with variance analysis, and optimize a skill's **description** for better triggering accuracy.

## Triggers
"create / build a skill", "edit / optimize this skill", "test / eval / benchmark a skill".

## Note
This is the skill you'd use to actually build **ask-razor** (see `ASK-RAZOR-PLAN.md`). Note: skills are saved/installed via **Settings → Capabilities**, not from inside a session.
