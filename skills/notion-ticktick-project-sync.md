# notion-ticktick-project-sync

**Group:** Personal & workflow

## What it does
Keeps your side-project list **in sync** between your Notion 🏗️ Project database and your TickTick 🔧Projects board. Any create / rename / status / priority / category / delete change in **either** app is mirrored to the other so both always match.

**Roles:**
- **Notion** — home for the project's document and insights.
- **TickTick** — quick capture on the go; holds a link back to the Notion page.

## Triggers
Creating, renaming, or changing status/priority/category of a project in either app; starting, planning, pausing, finishing, or abandoning a project — even if you only mention one app and never say "sync." Phrases like "add this project", "mark X as done / in progress", "I'm starting / parking / dropping a project", "new app idea", "update my project board".

## Pairs with
`dev-folder-organizer` (the local folder side of a project).
