# xlsx

**Group:** Document & output builders

## What it does
Create, edit, and analyze spreadsheets (`.xlsx`, `.xlsm`, `.xltx`, `.csv`, `.tsv`): formulas, formatting, charts, data analysis, and cleaning messy tabular data into proper sheets.

## Triggers
"Excel", "spreadsheet", ".xlsx", "data table", "budget", "financial model", "chart/graph", "tabular data" — especially when a spreadsheet file is referenced by name.

## Not for
Tasks where the real deliverable is a Word doc, HTML report, standalone script, or Google Sheets integration.

## Workflow note
Format skill — collect the data first, then load this to build the sheet.
