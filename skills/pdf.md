# pdf

**Group:** Document & output builders

## What it does
A full PDF toolkit: read/extract text and tables, create new PDFs, merge and split, rotate pages, watermark, fill forms, encrypt/decrypt, extract images, and OCR scanned PDFs to make them searchable.

## Triggers
"PDF", ".pdf", "form", "extract", "merge", "split" — or any request to produce or process a `.pdf`.

## Workflow note
Format skill — gather content first, then load this to build the PDF.
