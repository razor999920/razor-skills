# pptx

**Group:** Document & output builders

## What it does
Anything involving PowerPoint files (`.pptx` / `.potx`) as input or output: creating slide/pitch decks, reading or extracting text from slides, editing existing decks, combining or splitting decks, and working with templates, layouts, speaker notes, or comments.

## Triggers
"deck", "slides", "presentation", or any `.pptx`/`.potx` filename — regardless of what happens to the content afterward.

## Workflow note
Format skill — gather the substance first, then load this to build the deck.
