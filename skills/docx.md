# docx

**Group:** Document & output builders

## What it does
Create, read, edit, and manipulate Word documents (`.docx`) and templates (`.dotx`). Handles formatting like tables of contents, headings, page numbers, and letterheads; extracting or reorganizing content; inserting/replacing images; find-and-replace; and tracked changes or comments.

## Triggers
"Word doc", "word document", ".docx", ".dotx"; requests for a "report", "memo", "letter", or "template" as a Word file.

## Not for
PDFs, spreadsheets, Google Docs, or coding tasks unrelated to document generation.

## Workflow note
This is a **format** skill — research/gather content first, then load this to build the document.
