# create-cowork-plugin

**Group:** Skill & plugin authoring

## What it does
Guides you through creating a **new plugin** from scratch in a Cowork session — bundling skills, MCPs, and tools — and delivers the final `.plugin` file. Requires Cowork mode with access to the outputs directory.

## Triggers
"create / build / make / develop / scaffold / start / design a plugin".

## Note
A plugin is the distributable container; a skill is one ingredient inside it. Use this when you want to package and share a set of skills.
