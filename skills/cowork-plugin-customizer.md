# cowork-plugin-customizer

**Group:** Skill & plugin authoring

## What it does
Customizes an existing Claude Code plugin for a specific organization's tools and workflows — adjusting connectors, skills, and settings.

## Triggers
"customize / set up / configure / tailor / tweak / modify a plugin", "customize plugin connectors / skill / settings".

## Pairs with
`create-cowork-plugin` (build new) vs. this (tailor existing).
